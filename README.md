# UNLOCK - NFT Marketplace

A neo-brutalist NFT Marketplace prototype built with React, Firebase, and Gemini AI.

## Features
- **Utility-First NFTs**: Access backstage passes, game items, and creator content.
- **Neo-Brutalist Design**: High-contrast, bold aesthetics with Day/Night themes.
- **AI Vibe Check**: Gemini AI analysis of NFT drops.
- **Gamification**: XP and level-up system for user engagement.
- **Creator Studio**: Tools for creators to launch drops and reward fans.
- **Verifier Tool**: QR-based logic for real-world utility validation.

## Tech Stack
- Frontend: React, Tailwind CSS, Lucide React
- Backend: Firebase (Auth & Firestore)
- AI: Google Gemini Flash
